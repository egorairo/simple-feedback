# Simple Feedback

SimpleFeedback simple feedback tool for newsletters & websites (ReactJS, NodeJS, MongoDB, ExpressJS)

## Features

- ReactJS
- NodeJS
- MongoDB
- ExpressJS

## How to run the project

1. `git clone https://github.com/egorairo/jobster-web.git`

2. `cd server`

2. `npm i`

3. `npm run dev`

4. `cd client`

5. `npm i`

6. `npm run dev`
